## Project Overview
This project addresses the challenge of analyzing hospital admission trends to optimize resource allocation and improve patient care efficiency. It provides an automated pipeline to process raw admission records and generate actionable insights for hospital administrators. The output is a set of reproducible analysis notebooks and clean data summaries.
